    • Mr-Cap: Magnetic Resonance Connectome Automated Pipeline

    an end-to-end pipeline for ingesting, processing, and analyzing diffusion MRI data

    Project maintained by openconnectome Hosted on GitHub Pages — Theme by mattgraham

    This project contains several tools to play with fiber data in MRI Studio format [ http://www.mristudio.org/ ].

    The mrcap subdirectory contains the source code that generates graphs from dMRI streamlines. It produces two varieties of output:

    • smallgraphs: these use brain atlas regions of interest (70) to make 70z70 matrixes
    • biggraphs: these create connectivities among greymatter voxels.

    The cmapper directory contains simple tools to manipulate connectome mapper outputs.
